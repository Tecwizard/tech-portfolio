<ProjectCard
  title="TRP Interactive Map"
  link="https://map.terra-isles.com"
  bg="linear-gradient(to right, #009245 0%, #FCEE21 100%)"
>
  A custom interactive map for Terra Isles Roleplay built with the Leaflet library. The map provides landmark information as well as active and historic emergency call data. In the future, the map will also provide real-time information about emergency units and their status to dispatchers and other emergency personnel via the CAD/MDT system. The current version of the map is publicly available.
</ProjectCard>
